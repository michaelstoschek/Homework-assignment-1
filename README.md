# Homework-assignment-1

## Description
This repository contains a small standalone Python project called **Customer Request Prioritizer**.

The project provides simple utility functions for handling customer service requests. It can count open requests, filter urgent requests, and classify request priority.

## Functions
`count_open_requests(requests)`  
Returns the number of requests with status `"open"`.

`filter_urgent_requests(requests)`  
Returns all requests with priority `"high"`.

`classify_request(priority)`  
Classifies a request as `"urgent"` or `"normal"`.
